# Research archive scripts

These scripts were migrated from the local research workspace for traceability.
Some defaults may reference historical local experiments such as A5000,
p15_allcam, or output/local_feedback.

They are not the official A100 training entrypoints.

Use official migration-safe scripts in `scripts/` for server runs:

- `train.py`
- `launch_a100_experiments.py`
- `check_closed_loop_config.py`
- `validate_no_lidar_leakage.py`
- `validate_repair_safety.py`
- `collect_experiment_outputs.py`
- `evaluate.py`
- `evaluate_geometry_metrics.py`
- `render_periodic_panels.py`
